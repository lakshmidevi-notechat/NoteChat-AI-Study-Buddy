# NoteChat AI Study Buddy 📚🤖

AI-powered PDF Q&A tool using Streamlit and Gemini API. Upload your notes and ask questions in Tamil or English.

### ✨ Features
- 📄 Chat with your PDFs - Upload notes, get answers
- 🗣️ Supports Tamil + English 
- ⚡ Powered by Google Gemini + Streamlit
- 🎓 Perfect for students

### 🚀 Try it Live
**Live App:** Deploying to Streamlit Cloud soon!

### 🛠️ How to Run Locally
1. Clone: `git clone https://github.com/lakshmidevi-notechat/NoteChat-AI-Study-Buddy.git`
2. Install: `pip install -r requirements.txt`
3. Add Gemini API key in `.streamlit/secrets.toml`
4. Run: `streamlit run app.py`
